How to get a Google Gemini API Key?

Follow these steps to get your free API key:

  1. Go to Google AI Studio. You might need to sign in with your Google account.
  2. Click the "Create API key" button.
  3. You will be asked to create a new Google Cloud project if you don't already have one associated. Follow the on-screen instructions to create or select a project.
  4. Once the project is created (or if you already had one), your API key will be generated.
  5. Copy the API key shown on the screen. It's a long string of characters.
  6. Return to this settings page and paste the API key into the field above.
  7. Click the "Save Key" button.

Important! Treat your API key like a password. Do not share it publicly.
